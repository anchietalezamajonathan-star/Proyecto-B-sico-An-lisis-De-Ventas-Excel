# Proyecto-B-sico-An-lisis-De-Ventas-Excel

Proyecto: Análisi de Ventas Básico en Excel
Este proyecto consiste en el procesamiento y visualización de una base de datos con 5.000 registros de ventas (facturas de enero a diciembre), incluyendo fechas, clientes, productos (Audífonos, Laptops, Monitores, etc.), categorías (Accesorios, Oficina, Tecnología), cantidades, precios, vendedores y formas de pago (Efectivo, Sinpe, Tarjeta, Transferencia). 

Objetivos principales:
Organizar y limpiar datos comerciales de forma estructurada. 
Analizar el comportamiento de ventas por diferentes dimensiones. 

Herramientas y técnicas utilizadas:
Tablas Dinámicas para resumir muchos datos. 
Slicers para filtrado dinámico por mes, producto, vendedor, categoría y forma de pago. 
Gráficos avanzados: Línea de tendencia mensual, barras horizontales para ranking de productos/vendedores, pie chart para proporciones de categorías.
Cálculos clave con fórmulas (BDSUMA, BDPROMEDIO, BDMAX, BDMIN y BDCONTARA).

Insights obtenidos:

Tecnología representa ~35% de las ventas totales. Luis lidera con 852.583 € (21,13% del total), seguido de cerca por Ana. 
Pico de ventas en junio (405.588 €); mes más bajo: abril (288.843 €). 
Sinpe móvil representa el 25,72% de las formas de pago
